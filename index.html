Meus trabalhos recentes com APIs: um tour pelo meu portfólio

Eu criei este site para treinar minhas habilidades de consumo de APIs (Interfaces de Programação de Aplicação). As APIs são ferramentas que permitem a comunicação entre diferentes sistemas, possibilitando o compartilhamento de dados e a realização de tarefas de forma automatizada.

API sobre cotação do dólar:


Fornece informações atualizadas sobre o valor do dólar em relação à moeda nacional para ajudar em transações financeiras e comerciais.

API de CEP:


Encontra informações detalhadas sobre endereços no Brasil por código postal para validação e preenchimento de endereços em diferentes aplicativos.

API de Filmes:


Permite pesquisar informações detalhadas sobre filmes específicos, incluindo título, gênero e sinopse.

API de temperatura:


Fornece informações precisas sobre as condições climáticas atuais de diferentes locais em todo o mundo
